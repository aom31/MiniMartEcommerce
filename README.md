# MiniMartEcommerce
on backend side create project ecommerce for user product and order with golang


Feature:

    - Authentication
        - SignUp
        - Login
        #use JWT, Bcrypt
        
    - User-Management
     - Get Profile user
     - Get Order History own
     
    - Product-Management
        - Get all products list
        - Get detail product
        
    - Order-management
        - Create saleorder
        - Cancel saleorder
        - Get detail saleorder
     # gin, gorm

Stack:

    - Golang
    - Gin, Gorm, GRPC
    - Mongodb, Mysql
    - Docker , Kubernetes
    - Hexagonal Architecture
